## Как запускать

В репозитории встречаются два основных типа проектов:

1) Консольные учебные задачи (.pas)
- Можно компилировать и запускать с помощью Free Pascal (FPC) или среды Lazarus.
- Способ: открыть файл .pas в Lazarus, создать проект или собрать через fpc в терминале.

2) Визуальные приложения (Delphi/Lazarus)
- Содержат формы (.dfm/.lfm), ресурсы (изображения .bmp/.gif/.ico, звуки), проектные файлы (.lpi/.lpr, .dpr/.dproj, .lps).
- Рекомендуется открывать в Lazarus (кроссплатформенно) или в Embarcadero Delphi (Windows).

Примеры:
- Проект «Графики функций»: открыть соответствующий проектный файл (например, .lpi/.lpr или .dpr) в Lazarus/Delphi и собрать.
- «SchoolFight»: проект с ресурсами (папки img, snd); запуск в подходящей среде на Windows наиболее предсказуем.

## Требования и примечания

- Язык исходников: Pascal (FreePascal/Delphi). Для Delphi-проектов предпочтительна сборка под Windows.
- Кодировки: встречаются кириллические имена и возможные файлы в Windows-1251. При проблемах отображения в Lazarus/IDE попробуйте переключить кодировку файла на UTF-8/CP1251.
- Пути с пробелами и кириллицей: современные IDE поддерживают, но на старых системах возможны сложности. Рекомендуется открывать проекты непосредственно из IDE, чтобы она корректно нашла ресурсы.
- Ресурсы: многие проекты зависят от относительно расположенных папок (img, sound, picture, snd и т.п.). Не изменяйте структуру папок, чтобы ресурсы находились по ожидаемым путям.
