# График напряга

Небольшое десктоп?приложение для визуализации вашей учебной нагрузки на неделю: учитываются ответы на уроках, проверочные и контрольные работы. Перед началом вы можете указать личные предпочтения по предметам, чтобы получить более точный и детальный график, а также гибко настроить внешний вид.

## Возможности

- Сохранение расписания, чтобы не вводить его заново при следующем запуске.
- Отображение предметов при наведении на график (подсказки по точкам/отрезкам).
- Настройка цветов для осей, графика и разметки (сеток, делений и т. п.).
- Мгновенное обновление графика при изменении параметров — изменения видны сразу.

## Запуск

- Исполняемый файл находится в корневой папке проекта:
  - Скачать/запустить: [BusynessChart.exe](./BusynessChart.exe)
- Поддерживаемая платформа: Windows.

## Как это работает

1. Укажите расписание и предстоящие активности (ответы, проверочные, контрольные).
2. Задайте предпочтения по предметам (влияние на «вес»/важность и итоговый график).
3. Настройте внешний вид:
   - цвета осей,
   - цвет графика,
   - цвет и плотность разметки (сетки, деления по дням недели),
   - размещение/масштаб.
4. Наводите курсор на точки/участки — увидите подсказки с названиями предметов и деталями.
5. Сохраните расписание, чтобы при следующем запуске просто продолжить работу.

## Сохранение и загрузка

- Сохранение: используйте пункт меню/кнопку «Сохранить расписание». Будет предложено указать место и имя файла.
- Загрузка: используйте «Открыть расписание», чтобы вернуться к ранее сохранённым данным.

## Печать

- Готовый график можно распечатать из приложения или сохранить в файл и распечатать позже.

## Примечания

- При первом запуске настройте цвета и параметры под свои предпочтения — все изменения применяются к графику мгновенно.
- Если подсказки не видны, убедитесь, что вы наводите курсор непосредственно на элементы графика (точки/отрезки).
