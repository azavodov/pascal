# Физика. Обучающая и тестирующая система

## Краткое описание
Приложение помогает в удобной форме изучать термины из школьного курса физики и проходить тестирование. Определения рассортированы по разделам, есть быстрый поиск и возможность проходить тесты с вводом собственного ответа. Алгоритм сравнения анализирует введённый текст и выдаёт процент совпадения с эталонным определением, а также вердикт о правильности.

## Ключевые возможности
- База данных расширена до 460 определений.
- Поиск по базе определений.
- Тестирование без выбора вариантов: вы вводите своё определение.
- Алгоритм сравнения оценивает процент совпадения и определяет корректность ответа.
- Можно одновременно сравнивать несколько результатов.
- Все результаты шифруются; расшифровка доступна только внутри программы.
- Возможность составлять собственные тесты: добавляйте вопросы из базы или создавайте свои.
- Настройка внешнего вида: выбор шрифта и размера текста.

- Документация/описание: ./ОПИСАНИЕ.docx

## Как пользоваться
1. Откройте исполняемый файл (рекомендуется новая версия: PhysicsTests_5.5.exe).
2. Изучайте определения по разделам или воспользуйтесь поиском.
3. Для тестирования выберите раздел и отвечайте своими словами — система сравнит ваш ответ с эталоном.
4. При необходимости создайте собственный тест и добавляйте вопросы.
5. Сравнивайте результаты между собой; итоги шифруются и доступны к просмотру только в программе.

## Требования и совместимость
- Windows: запуск .exe файлов из корня каталога.
- Другие ОС: возможен запуск через эмуляторы/совместимые среды (например, Wine), но официально не тестировалось.
